# 🚗 CarTunisia Agency

A modern and responsive car rental website built using HTML, CSS, and JavaScript.

## 🌍 Overview

CarTunisia Agency is a simple front-end project that showcases a car rental service in Tunisia.
It includes a clean design, responsive layout, and a contact form with validation.

## ✨ Features

*  Car fleet showcase (Hyundai, Renault, BMW, Audi...)
*  Fully responsive design (mobile friendly)
*  Contact form with JavaScript validation
*  Mobile navbar toggle menu
*  Modern UI with dark theme

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)
